Este projeto é um gerador de currículo online simples. Ele permite que os usuários insiram seus dados pessoais, experiência profissional e formação acadêmica e gera um currículo em PDF.

Funcionalidades
Inserção de dados pessoais (nome, data de nascimento, endereço, contato)
Inserção de experiência profissional (empresa, função, datas de início e fim)
Inserção de formação acadêmica (instituição, curso, datas de início e fim)
Geração de um currículo em PDF


Tecnologias Utilizadas:

HTML
CSS (com Bootstrap)
JavaScript (com jsPDF)
